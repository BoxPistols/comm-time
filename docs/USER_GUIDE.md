## Comm Timeとは？

Comm Timeは、ミーティングや作業時間を効率的に管理するためのタイマーアプリです。

### 主な機能

✅ **ミーティングタイマー** - 会議時間を管理し、決まった時間にアラームでお知らせ
✅ **ポモドーロタイマー** - 作業と休憩を繰り返す時間管理術をサポート
✅ **TODO管理** - やるべきことをリストで管理
✅ **メモ機能** - 会議や作業のメモを記録
✅ **クラウド保存** - ログインすると、データをクラウドに保存して他のデバイスと同期
✅ **ダークモード** - 目に優しい暗い画面モード
✅ **カスタマイズ** - アラーム音、通知、バイブレーションを自由に設定

## ミーティングタイマー

会議やミーティングの時間管理に最適なタイマーです。

### 使い方

#### 1. アラームポイントを設定

アラームポイントとは、「○○分後にアラームを鳴らす」という設定です。

**例**: 30分会議の場合
- 25分後: 「あと5分です」のアラーム
- 30分後: 「終了時刻です」のアラーム

**設定方法**:
1. 「アラームポイント追加」ボタンをクリック
2. 分数を入力（例: 25）
3. 「追加」をクリック
4. 必要に応じて複数追加

#### 2. タイマーをスタート

1. ▶️「スタート」ボタンをクリック
2. タイマーが動き始めます
3. 設定した時間になるとアラームが鳴ります

#### 3. アラームを停止

アラームが鳴ったら：
- 画面をタップ
- または「アラーム停止」ボタンをクリック

### カウントダウンモード

特定の時刻までの残り時間を表示できます。

**使い方**:
1. 設定 ⚙️ をクリック
2. 「カウントダウンモード」をON
3. 目標終了時刻を設定（例: 15:00）
4. タイマーが「あと○分○秒」と表示されます

## ポモドーロタイマー

作業と休憩を繰り返す「ポモドーロ・テクニック」を実践できます。

### ポモドーロ・テクニックとは？

25分間の集中作業 → 5分間の休憩 を繰り返す時間管理術です。

```
作業 25分 → 休憩 5分 → 作業 25分 → 休憩 5分 ...
```

4サイクル後に長めの休憩（15-30分）を取ることが推奨されています。

### 使い方

#### 1. 設定をカスタマイズ

1. 設定 ⚙️ → 「ポモドーロタイマー設定」
2. 以下を調整：
   - **作業時間**: 25分（デフォルト）
   - **休憩時間**: 5分（デフォルト）
   - **サイクル数**: 4回（デフォルト）
   - **無限モード**: ON にすると永遠に繰り返します

#### 2. タイマーをスタート

1. 「ポモドーロタイマー」タブをクリック
2. ▶️「スタート」ボタンをクリック
3. 作業に集中！

#### 3. アラームで切り替え

- **作業終了時**: アラームが鳴り、自動的に休憩タイマーに切り替わります
- **休憩終了時**: アラームが鳴り、次の作業タイマーに切り替わります

### ヒント

💡 作業中はTODOリストで「今やること」を選んでおくと集中しやすい
💡 休憩中はしっかり休んで、次の作業に備えましょう
💡 4サイクル後は長めの休憩を取ることをお勧めします

## TODO管理

やるべきことをリストで管理できます。

### 基本操作

#### TODO を追加

1. TODOセクションの入力欄にテキストを入力
2. 「+」ボタンをクリック
3. リストに追加されます

#### TODO を完了

- ☐ チェックボックスをクリック
- ☑ になり、完了したことを示します

#### TODO を編集

1. ✏️ 編集アイコンをクリック
2. テキストを修正
3. ✓ 保存アイコンをクリック

#### TODO を削除

- ✕ 削除アイコンをクリック

### 便利な機能

#### 期限を設定

1. 📅 カレンダーアイコンをクリック
2. 日付と時刻を設定
3. 期限が近づくと色が変わります：
   - 🟢 **緑**: まだ余裕があります
   - 🟡 **黄**: 期限が近づいています（24時間以内）
   - 🔴 **赤**: 期限を過ぎています

#### ドラッグ&ドロップで並び替え

TODOをドラッグして、好きな順番に並び替えられます。

#### ミーティングとポモドーロで別々に管理

- **ミーティングタブ**: 会議用のTODO
- **ポモドーロタブ**: 作業用のTODO

それぞれ独立しているので、用途に応じて使い分けられます。

## クラウド保存機能（新機能）

ログインすると、TODO・メモをクラウドに保存できます。

### できること

✅ **複数デバイスで同期** - スマホとPCでデータを共有
✅ **データのバックアップ** - クラウドに安全に保存
✅ **リアルタイム同期** - 変更がすぐに他のデバイスに反映

### 使い方

#### 1. アカウント作成（初回のみ）

1. 画面右上の「ログイン」ボタンをクリック
2. 「アカウントをお持ちでない方はこちら」をクリック
3. メールアドレスとパスワード（6文字以上）を入力
4. 「新規登録」をクリック
5. 確認メールが届きます
6. メール内のリンクをクリックしてアカウントを有効化

#### 2. ログイン

1. 「ログイン」ボタンをクリック
2. メールアドレスとパスワードを入力
3. 「ログイン」をクリック
4. ヘッダーにメールアドレスが表示されます

#### 3. クラウド保存を有効化

1. ログイン後、ヘッダーに表示される 💾 アイコンをクリック
2. アイコンが緑色になればON（クラウド保存有効）
3. グレーの場合はOFF（ローカル保存）

#### 4. データの切り替え

| モード | 保存先 | 同期 | 説明 |
|-------|-------|------|------|
| **ログインなし** | ブラウザ内 | ✕ | 従来通りの使い方 |
| **ログイン + OFF** | ブラウザ内 | ✕ | ログインしているが保存はローカル |
| **ログイン + ON** | クラウド | ✅ | データをクラウドに保存・同期 |

### よくある質問

**Q: ログインしないと使えませんか？**
A: いいえ。ログインなしでも従来通り使えます。クラウド保存は任意の機能です。

**Q: 既存のTODO/メモはどうなりますか？**
A: ログインしても、既存のデータはブラウザ内に残ります。クラウド保存をONにすると、新しく作成したデータがクラウドに保存されます。

**Q: データは安全ですか？**
A: はい。各ユーザーのデータは完全に分離されており、他の人はアクセスできません。

**Q: 無料ですか？**
A: はい。クラウド保存機能は完全無料です。

## 設定のカスタマイズ

画面右上の ⚙️ 設定ボタンから、さまざまな設定を変更できます。

### アラーム音の設定

#### ミーティングタイマー

- **音量**: 0〜100 で調整
- **周波数**: 100〜1000Hz で音の高さを調整
- **試聴**: 「音をテスト」ボタンで確認

#### ポモドーロタイマー

作業アラームと休憩アラームを別々に設定できます：

- **作業アラーム**: 作業終了時の音
- **休憩アラーム**: 休憩終了時の音

それぞれ音量と周波数を調整できます。

### 通知設定

| 設定 | 説明 |
|------|------|
| **通知** | ブラウザ通知を有効にする（初回は許可が必要） |
| **バイブレーション** | スマホでバイブレーションを有効にする |
| **フラッシュ** | アラーム時に画面を点滅させる |
| **チクタク音** | タイマー動作中に秒針音を鳴らす |

### ダークモード

画面を暗くして目の疲れを軽減します。

**切り替え方法**:
- 設定 → ダークモード ON/OFF
- または画面右上の 🌙/☀️ アイコンをクリック

### 設定のリセット

「現在の設定を初期値にリセット」ボタンで、すべての設定を初期状態に戻せます。

## よくある質問（FAQ）

### 一般的な質問

**Q: スマホでも使えますか？**
A: はい。スマホのブラウザで使えます。ホーム画面に追加すればアプリのように使えます。

**Q: オフラインでも使えますか？**
A: はい。一度ページを開けば、オフラインでも基本機能は使えます。ただし、クラウド保存の同期はオンライン時のみです。

**Q: データは消えませんか？**
A: ブラウザのデータを削除しない限り、ローカルデータは残ります。クラウド保存を使えばさらに安全です。

### タイマーについて

**Q: アラームが鳴りません**
A: 以下を確認してください：
1. デバイスの音量が0になっていませんか？
2. ブラウザがミュートになっていませんか？
3. 設定でアラーム音量が0になっていませんか？

**Q: 通知が表示されません**
A: ブラウザの通知許可が必要です：
1. 通知ボタンをクリック
2. ブラウザの許可ダイアログで「許可」をクリック

**Q: バックグラウンドで動きますか？**
A: はい。タブを閉じなければ、他の作業をしていてもタイマーは動きます。

### TODOについて

**Q: TODOの順番を変えられますか？**
A: はい。TODOをドラッグ&ドロップで並び替えられます。

**Q: 完了したTODOを消したい**
A: ✕ ボタンで削除できます。完了済みを一括削除する機能は今後追加予定です。

**Q: ミーティングとポモドーロのTODOは共有されますか？**
A: いいえ。それぞれ別々に管理されます。

### クラウド保存について

**Q: ログインせずに使えますか？**
A: はい。ログインなしでも従来通り使えます。

**Q: 確認メールが届きません**
A: 迷惑メールフォルダを確認してください。それでも届かない場合は、別のメールアドレスをお試しください。

**Q: パスワードを忘れました**
A: 現在、パスワードリセット機能は準備中です。別のアカウントを作成してください。

**Q: アカウントを削除したい**
A: ログアウトすれば、クラウドデータにアクセスできなくなります。完全削除をご希望の場合はお問い合わせください。

## トラブルシューティング

### タイマーが動かない

**症状**: スタートボタンを押してもタイマーが動かない

**解決方法**:
1. ページをリロード（F5キー）
2. ブラウザのキャッシュをクリア
3. 別のブラウザで試す

### データが消えた

**症状**: TODOやメモが消えている

**考えられる原因**:
- ブラウザのデータ削除
- プライベートモード（シークレットモード）で使用
- 別のブラウザ/デバイスで開いている

**解決方法**:
- クラウド保存を使用していれば、ログインで復元できます
- ローカル保存の場合は、ブラウザのデータを削除しないよう注意

### アラーム音が小さい/大きい

**解決方法**:
1. 設定 ⚙️ をクリック
2. 「アラーム音設定」で音量を調整
3. 「音をテスト」で確認

### 画面が真っ白/エラーが表示される

**解決方法**:
1. ページをリロード
2. ブラウザのコンソールでエラーを確認
3. ブラウザを最新版に更新
4. 別のブラウザで試す

### それでも解決しない場合

以下の情報を添えてお問い合わせください：
- 使用しているブラウザとバージョン
- 使用しているデバイス（PC、スマホなど）
- エラーメッセージ（あれば）
- 発生した状況

## ヒントとコツ

### 効率的な使い方

**ミーティングで**:
- 開始前にアジェンダをTODOに入力
- 議論した内容をメモに記録
- アラームで時間管理

**作業で**:
- ポモドーロタイマーで集中力UP
- 作業ごとにTODOを作成
- 完了したらチェック✓

**チーム利用**:
- プロジェクターに映して共有
- タイマーをみんなで確認
- 時間を見える化

### カスタマイズのおすすめ

**静かな環境で**:
- バイブレーションのみON
- アラーム音量を小さく
- フラッシュをON

**集中したい時**:
- ポモドーロタイマー使用
- 通知はすべてOFF
- ダークモードでリラックス

**会議で**:
- 複数のアラームポイント設定
- カウントダウンモード使用
- 通知とフラッシュをON
