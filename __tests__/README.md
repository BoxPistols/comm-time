# テストについて

## 概要

このプロジェクトでは、コードの品質と信頼性を保証するためにテストを実装しています。

## テストファイル

### `default-values.test.ts`

初期値定数と数値入力検証のテストです。

**テストカバレッジ:**
- 初期値定数の一貫性確認
- 数値入力の検証ロジック（NaN対策）
- 最小値制限の動作確認

## テストの実行

### 必要な依存関係のインストール

```bash
npm install --save-dev jest @jest/globals @testing-library/react @testing-library/jest-dom jest-environment-jsdom
```

### テストの実行方法

```bash
# すべてのテストを実行
npm test

# 特定のテストファイルを実行
npm test default-values

# カバレッジレポート付きで実行
npm test -- --coverage

# ウォッチモードで実行
npm test -- --watch
```

## テストの重要性

このプロジェクトでは、以下の点を重視してテストを実装しています：

1. **数値入力の堅牢性**: ユーザーが空文字列や無効な値を入力した場合でも、アプリケーションがクラッシュしないことを保証
2. **初期値の一貫性**: マジックナンバーを排除し、定数から初期値を参照することで保守性を向上
3. **最小値制限**: 各入力フィールドの最小値が正しく機能することを確認
