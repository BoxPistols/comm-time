## 🎯 機能一覧

### ミーティングタイマー
- ⏱️ **経過時間モード**: 会議開始からの経過時間を計測
- ⏰ **カウントダウンモード**: 終了時刻を設定して残り時間を表示
- 🔔 **複数アラームポイント**: 30分、50分、60分など任意の時間でアラーム
- 📝 **リアルタイムメモ**: 会議中の議事メモを記録
- ✅ **TODOリスト**: 会議で決まったタスクを管理

### ポモドーロタイマー
- 🎯 **作業時間/休憩時間の自動切り替え**: 25分作業→5分休憩を自動で繰り返し
- 🔄 **サイクルカウンター**: 完了したサイクル数を表示
- ♾️ **無限モード**: 手動で停止するまで永続的に動作
- 🔔 **異なるアラーム音**: 作業開始と休憩開始で別の音で通知
- 📝 **作業記録**: 何をしたか記録できるメモとTODO機能

### 共通機能
- 🔊 **チクタク音**: タイマー動作中の秒針音（ON/OFF可能）
- 📳 **バイブレーション**: アラーム時にスマホが振動（対応デバイスのみ）
- ⚡ **フラッシュアラート**: 画面全体を白く点滅させて視覚的に通知
- 🔔 **ブラウザ通知**: バックグラウンドでもアラームを通知
- 💾 **自動保存**: すべての設定とデータを自動的に保存（ブラウザを閉じても保持）

## 📖 ユーザー向け使い方マニュアル

### 初めて使う方へ

#### ステップ1: アプリにアクセス

1. Webブラウザで [https://comm-time.vercel.app](https://comm-time.vercel.app) を開く
2. 画面上部に「Comm Time」と表示され、現在時刻が表示されていることを確認
3. 2つのタブが表示されます：
   - 「ミーティングタイマー」（会議用）
   - 「ポモドーロタイマー」（作業用）

#### ステップ2: 通知の設定（推奨）

1. 画面右上の 🔔 ボタンをクリック
2. ブラウザから通知許可を求められたら「許可」をクリック
3. ボタンが青色に変わったら設定完了
4. これでバックグラウンドでもアラームを受け取れます

**注意**: iOS Safariではブラウザ通知が使えません。代わりにフラッシュやバイブレーションをご利用ください。

### ミーティングタイマーの使い方

会議や打ち合わせの時間管理に最適です。

#### 基本的な使い方

##### シナリオ1: 60分の会議で時間管理したい

**手順:**

1. **「ミーティングタイマー」タブをクリック**
   - 画面左側のタブが選択されていることを確認

2. **アラームポイントを確認**
   - デフォルトで 30分、50分、60分 が設定済み
   - そのまま使う場合はこのステップをスキップ

3. **タイマーを開始**
   - 大きな「開始」ボタン（緑色）をクリック
   - タイマーの表示が「00:00:00」から動き始めます
   - 現在時刻の下に「開始時刻」と「終了予定時刻」が表示されます

4. **会議を進行**
   - 30分経過時: アラームが鳴り、画面が点滅（「30分経過しました」と表示）
   - 50分経過時: 再びアラーム（「50分経過しました」と表示）
   - 60分経過時: 最後のアラーム（「60分経過しました」と表示）

5. **アラームを停止する方法**
   - 画面をタップ（フラッシュがONの場合）
   - または「アラーム停止」ボタンをクリック
   - 30秒経過すると自動的に停止します

6. **タイマーを一時停止**
   - 「一時停止」ボタン（赤色）をクリック
   - タイマーが止まります
   - 再度「開始」をクリックすると再開

7. **タイマーをリセット**
   - 「リセット」ボタン（グレー）をクリック
   - タイマーが 00:00:00 に戻ります
   - アラームポイントも初期状態にリセットされます

##### シナリオ2: 15時30分に終わりたい（カウントダウンモード）

**手順:**

1. **「ミーティングタイマー」タブを選択**

2. **カウントダウンモードをONにする**
   - 「カウントダウンモード」のトグルをクリック
   - 青色に変わったらON状態

3. **終了時刻を設定**
   - 時刻入力フィールドをクリック
   - 例: `15:30` と入力
   - 現在時刻が13:00なら、残り時間「02:30:00」と表示されます

4. **タイマーを開始**
   - 「開始」ボタンをクリック
   - 残り時間がカウントダウンされます

5. **0になったとき**
   - アラームが鳴ります
   - 「時間になりました！」とメッセージが表示
   - タイマーは自動的に停止します

#### アラームポイントのカスタマイズ

##### アラームポイントを追加する

**例: 15分、45分、75分でアラームを鳴らしたい**

**手順:**

1. 「アラームポイント」セクションを見つける
2. 既存のポイントを編集：
   - 最初のポイントの数値「30」をクリック
   - `15` と入力
3. 2番目のポイントを編集：
   - 「50」を `45` に変更
4. 3番目のポイントを編集：
   - 「60」を `75` に変更
5. 設定は自動的に保存されます

##### アラームポイントを追加する

**手順:**

1. 「アラームポイントを追加」ボタンをクリック
2. 新しいポイントが追加されます（現在の経過時間+1分）
3. 数値を編集して任意の時間に設定
4. 例: `90` と入力すると90分（1時間30分）でアラーム

##### アラームポイントを削除する

**手順:**

1. 削除したいポイントの右側にある「×」ボタンをクリック
2. そのポイントが削除されます
3. 最低1つはポイントを残すことをおすすめします

#### アラーム音のカスタマイズ

##### 音量を調整する

**手順:**

1. 「ミーティングアラーム設定」セクションまでスクロール
2. 「音量」のスライダーを左右にドラッグ
   - 左に動かす: 音量が小さくなる
   - 右に動かす: 音量が大きくなる
3. 数値で確認（0〜100）
4. 推奨値: 40〜60

##### 周波数を変更する

**手順:**

1. 「周波数」の数値入力フィールドをクリック
2. 数値を入力:
   - 低い音が好み: `200` 〜 `400` Hz
   - 中くらいの音: `400` 〜 `600` Hz
   - 高い音が好み: `600` 〜 `800` Hz
3. デフォルト: `340` Hz

##### テスト再生

**手順:**

1. 「テスト」ボタン（黄色）をクリック
2. アラームが鳴ります（5秒ごとに最大30秒間）
3. 停止したい場合：
   - 画面をタップ
   - または「アラーム停止」ボタンをクリック
4. 音が気に入らない場合は音量や周波数を調整して再テスト

##### 設定をリセット

**手順:**

1. 「リセット」ボタンをクリック
2. 音量: 44、周波数: 340 Hz に戻ります

### ポモドーロタイマーの使い方

集中して作業したいときに最適です。作業時間と休憩時間を自動で切り替えます。

#### 基本的な使い方

##### シナリオ: 標準的なポモドーロテクニック（25分作業×4セット）

**手順:**

1. **「ポモドーロタイマー」タブをクリック**

2. **設定を確認**
   - デフォルト設定:
     - 作業時間: 25分
     - 休憩時間: 5分
     - サイクル数: 4回
   - このまま使う場合は次のステップへ

3. **タイマーを開始**
   - 「開始」ボタン（緑色）をクリック
   - 作業時間が始まります（青い背景）
   - 画面に「🎯 作業時間」と表示

4. **25分経過後**
   - アラームが鳴ります（作業時間終了のお知らせ）
   - 自動的に休憩時間に切り替わります（オレンジ色の背景）
   - 画面に「☕ 休憩時間」と表示
   - メッセージ: 「お疲れ様です！休憩時間です」

5. **5分経過後（休憩終了）**
   - アラームが鳴ります（休憩終了のお知らせ）
   - 自動的に作業時間に戻ります（青い背景）
   - メッセージ: 「休憩終了！作業を開始してください」
   - サイクルカウンターが「1 / 4」に増えます

6. **4サイクル完了後**
   - タイマーが自動的に停止します
   - お疲れ様でした！

7. **途中で止めたい場合**
   - 「一時停止」ボタン（赤色）をクリック
   - 再開する場合は「開始」ボタン

8. **最初からやり直す場合**
   - 「リセット」ボタンをクリック
   - サイクルカウンターが 0 に戻ります

#### ポモドーロ設定のカスタマイズ

##### 作業時間を変更する

**例: 50分集中したい場合**

**手順:**

1. 「ポモドーロ設定」セクションを見つける
2. 「作業時間」の数値フィールドをクリック
3. `50` と入力
4. 設定が保存されます
5. 次回のタイマー開始から50分作業になります

##### 休憩時間を変更する

**例: 10分休憩したい場合**

**手順:**

1. 「休憩時間」の数値フィールドをクリック
2. `10` と入力
3. 設定が保存されます

##### サイクル数を変更する

**例: 8サイクル（丸1日作業）したい場合**

**手順:**

1. 「サイクル数」の数値フィールドをクリック
2. `8` と入力
3. 8サイクル完了後にタイマーが自動停止します

##### 無限モードを使う

**手順:**

1. 「無限モード（サイクル数無制限）」のチェックボックスをクリック
2. チェックが入ると、サイクル数の入力がグレーアウトします
3. タイマーは永遠に作業と休憩を繰り返します
4. 手動で「一時停止」または「リセット」するまで続きます

#### アラーム音を個別に設定

ポモドーロタイマーでは、作業開始時と休憩開始時で異なる音を設定できます。

##### 作業時間アラームの設定

**手順:**

1. 「🎯 作業時間アラーム設定」セクションを見つける
2. 音量スライダーを調整（推奨: 60〜70）
3. 周波数を設定（デフォルト: 240 Hz）
   - 低めの音で落ち着いて作業開始をお知らせ
4. 「テスト」ボタンで確認

##### 休憩時間アラームの設定

**手順:**

1. 「☕ 休憩時間アラーム設定」セクションを見つける
2. 音量スライダーを調整（推奨: 30〜40）
   - 休憩は優しい音量がおすすめ
3. 周波数を設定（デフォルト: 740 Hz）
   - 高めの音で軽快に休憩開始をお知らせ
4. 「テスト」ボタンで確認

### 共通機能の使い方

ミーティングタイマーとポモドーロタイマーの両方で使える機能です。

#### メモ機能

会議の議事録や作業内容を記録できます。

**手順:**

1. 画面右側の「📝 メモ」セクションを見つける
2. テキストエリアをクリック
3. 自由にメモを入力
   - 例（ミーティング）:
     ```
     議題:
     - Q1の売上報告
     - 新商品の企画案

     決定事項:
     - 来週までに資料作成
     ```
   - 例（ポモドーロ）:
     ```
     今日のタスク:
     - バグ修正 #123
     - 仕様書レビュー
     - コードリファクタリング
     ```
4. 入力内容は自動的に保存されます
5. ブラウザを閉じても内容は保持されます
6. ミーティングとポモドーロで別々に保存されます

#### TODOリスト

タスク管理ができます。

##### TODOを追加する

**手順:**

1. 画面右側の「✅ TODOリスト」セクションを見つける
2. 「新しいTODOを入力...」フィールドをクリック
3. タスクを入力
   - 例: `資料を作成する`
4. Enterキーを押すか「追加」ボタンをクリック
5. リストに追加されます

##### TODOを完了にする

**手順:**

1. 完了したタスクの左側にあるチェックマークボタンをクリック
2. テキストに取り消し線が入り、緑色の背景になります
3. もう一度クリックすると未完了に戻ります

##### TODOを編集する

**手順:**

1. 編集したいTODOの右側にある鉛筆アイコンをクリック
2. テキストが入力可能になります
3. 内容を変更
   - 例: `資料を作成する` → `資料を作成してメール送信する`
4. 保存ボタン（チェックマーク）をクリック
5. または×ボタンでキャンセル

##### TODOを削除する

**手順:**

1. 削除したいTODOの右側にある×ボタンをクリック
2. 確認なしで即座に削除されます
3. 削除したTODOは復元できませんのでご注意ください

##### TODOの順序を変更する

**方法1: ボタンで移動**

**手順:**
1. 上に移動したい場合: ↑ボタンをクリック
2. 下に移動したい場合: ↓ボタンをクリック
3. 一番上や一番下では該当ボタンがグレーアウトします

**方法2: ドラッグ&ドロップ**

**手順:**
1. 移動したいTODOをマウスでドラッグ（長押し）
2. 好きな位置まで移動
3. マウスを離すとその位置に配置されます

#### ヘッダーの設定ボタン

画面右上に4つの設定ボタンがあります。

##### チクタク音 🔊

タイマー動作中に「チクタク」という秒針音を鳴らします。

**手順:**

1. 🔊 ボタンをクリック
2. 緑色: ON（音が鳴る）
3. グレー: OFF（音が鳴らない）
4. タイマーを開始すると秒針音が聞こえます

**ヒント**: 集中したいときはOFF、時間の経過を意識したいときはONがおすすめ

##### バイブレーション 📳

アラーム時にスマートフォンが振動します。

**手順:**

1. 📳 ボタンをクリック
2. 紫色: ON（振動する）
3. グレー: OFF（振動しない）
4. スマートフォンで効果があります

**注意**: パソコンでは振動しません

##### フラッシュ ⚡

アラーム時に画面全体が白く点滅します。

**手順:**

1. ⚡ ボタンをクリック
2. 黄色: ON（画面が点滅）
3. グレー: OFF（点滅しない）
4. アラーム時に画面全体が白くなり、中央に「⏰ TIME UP! ⏰」と表示されます
5. 画面をタップすると消えます

**ヒント**: 音だけでは気づかない場合に便利です

##### 通知 🔔

ブラウザ通知でアラームをお知らせします。

**手順:**

1. 🔔 ボタンを初めてクリック
2. ブラウザから通知許可を求められます
3. 「許可」を選択
4. 青色: ON（通知が届く）
5. グレー: OFF（通知が届かない）
6. テスト通知が表示されます: 「通知が有効になりました！」

**メリット**:
- 他のタブを見ていても通知が届く
- 画面を見ていなくても気づける
- デスクトップに通知ポップアップが表示される

**注意**: iOS Safariでは使用できません

#### アラームを停止する4つの方法

アラームが鳴っているときは、以下の方法で止められます:

**方法1: フラッシュ画面をタップ**
- フラッシュがONの場合のみ
- 画面の白い部分をどこでもタップ
- 即座に停止します

**方法2: アラーム停止ボタン**
- ヘッダーに赤い「アラーム停止」ボタンが表示されます
- このボタンをクリック
- 確実に停止したいときに便利

**方法3: 画面をクリック**
- フラッシュがOFFの場合
- 画面のどこでもクリック
- 停止します

**方法4: 自動停止**
- 30秒間放置
- 自動的に停止します
- 気づかなかった場合の安全策

### よくある質問（FAQ）

#### Q1: データはどこに保存されますか？

**A**: すべてのデータはブラウザのローカルストレージに保存されます。

- サーバーには一切送信されません
- プライバシーは完全に保護されます
- ブラウザのデータを消去するまで保持されます
- 別のブラウザやデバイスでは共有されません

#### Q2: データを削除するには？

**A**: ブラウザのデータを消去してください。

**Chrome/Edgeの場合:**
1. F12キーを押して開発者ツールを開く
2. 「Application」タブをクリック
3. 左側の「Local Storage」を展開
4. サイトのURLをクリック
5. 右クリック → 「Clear」を選択
6. ページをリロード（F5キー）

**Safari/Firefoxの場合:**
1. 開発者ツールを開く
2. 「Storage」タブを探す
3. Local Storageのデータを削除

#### Q3: スマートフォンで使えますか？

**A**: はい、完全対応しています。

- レスポンシブデザインで画面サイズに自動対応
- タッチ操作に最適化
- バイブレーション機能も利用可能
- ホーム画面に追加して使用できます

**iPhoneでホーム画面に追加する方法:**
1. Safariで開く
2. 画面下部の共有ボタンをタップ
3. 「ホーム画面に追加」を選択
4. アプリのようにアイコンから起動できます

#### Q4: オフラインでも使えますか？

**A**: 初回アクセス後はオフラインでも動作します。

- PWA（Progressive Web App）として動作
- ネット接続なしでも基本機能は利用可能
- データは全てローカルに保存されます

#### Q5: 通知が届きません

**A**: 以下を確認してください:

1. **通知ボタンが青色か確認**
   - グレーの場合は🔔ボタンをクリック

2. **ブラウザの通知許可を確認**
   - Chrome: 設定 → プライバシーとセキュリティ → サイトの設定 → 通知
   - サイトが「許可」になっているか確認

3. **OSの通知設定を確認**
   - Windows: 設定 → システム → 通知
   - macOS: システム環境設定 → 通知
   - ブラウザの通知が許可されているか確認

4. **iOS Safariの場合**
   - ブラウザ通知は使用できません
   - フラッシュとバイブレーションを使用してください

#### Q6: 音が鳴りません

**A**: 以下を試してください:

1. **デバイスの音量を確認**
   - ミュートになっていないか
   - 音量が0になっていないか

2. **ブラウザのタブがミュートされていないか確認**
   - タブのスピーカーアイコンを確認

3. **テストボタンで確認**
   - アラーム設定の「テスト」ボタンをクリック
   - 音が鳴るか確認

4. **音量設定を上げる**
   - アラーム設定の音量スライダーを80以上に

#### Q7: チクタク音が鳴りません（スマートフォン）

**A**: モバイルブラウザの仕様です。

- 多くのモバイルブラウザは自動音声再生を制限しています
- タイマー開始ボタンをタップした後は再生されます
- これはブラウザのセキュリティ機能です

#### Q8: アラームが30秒で止まってしまいます

**A**: これは仕様です。

- アラームは5秒ごとに鳴り、30秒後に自動停止します
- 延々と鳴り続けることを防ぐための設計です
- 変更したい場合は開発者マニュアルを参照してください

#### Q9: 複数のタブで同時に使えますか？

**A**: 使えますが注意が必要です。

- 各タブが独立して動作します
- データは最後に更新したタブの内容が保存されます
- 通常は1つのタブでの使用をおすすめします

#### Q10: データをバックアップするには？

**A**: 現在、エクスポート機能はありません。

- データはブラウザのローカルストレージに保存
- 重要なメモはコピー＆ペーストで別の場所に保存してください
- 将来的にエクスポート機能を追加予定です
